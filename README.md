# 💰 Money Wise

A smart and simple financial management application designed to help users track expenses, manage budgets, and make better financial decisions.

---

## 🚀 Features

- 📊 Track daily expenses and income  
- 💸 Categorize transactions  
- 📅 View weekly and monthly summaries  
- 📈 Analyze spending patterns  
- 🔐 Simple and user-friendly interface

## ⚙️ Installation & Setup

Clone the repository:

```

git clone [https://github.com/javinarora05/Money-Wise.git](https://github.com/javinarora05/Money-Wise.git)

```

Navigate to the project folder:

```

cd Money-Wise

```

Install dependencies:

```

npm install

```

Run the project:

```

npm start

```
